# Simple Booking App

A simple React web app to view and book available time slots. Bookings are saved using localStorage.

## Features
- View available slots
- Book a time slot
- Prevent double bookings
- Confirmation screen
- Responsive design with Tailwind CSS
- Toast notifications for feedback

## Tech Stack
- React
- React Router
- Tailwind CSS
- React Hot Toast
- LocalStorage

## Run Locally
```bash
git clone <repo-url>
cd simple-booking-app
npm install
npm run dev
```

## Deployment
Deploy easily using Vercel or Netlify.
